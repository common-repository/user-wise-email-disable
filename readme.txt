== Description ==
This plugin is useful for **disable user wise emails**. You need to **drag and drop the user** and save it then this will work automatically.

== Installation ==
#### From within WordPress

1. Visit \'Plugins > Add New\'
2. Search for \'User Wise Email Disable\'
3. Activate \'User Wise Email Disable\' for WordPress from your \'Plugins\' menu from WordPress.

#### Manually

1. Upload the user-wise-email-disable/ folder to the /wp-content/plugins/ directory. 
2. Activate the plugin through the \'Plugins\' menu in WordPress.
